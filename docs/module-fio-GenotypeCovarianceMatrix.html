Class: GenotypeCovarianceMatrix

fio~GenotypeCovarianceMatrix(matrix, variants, positions)

Class for storing genotype covariance matrices.

Assumptions:
  • Covariances should be oriented towards the minor allele.
  • Variances on the diagonal are absolute values (they are not directional.)

Constructor

new GenotypeCovarianceMatrix(matrix, variants, positions)

Parameters:
Name Type Description
matrix Array.<Array.<number>> Pre-constructed matrix. Usually generated by the extractCovariance function in fio.
variants Map Map of variants -> matrix position. Variants should be chr:pos_ref/alt format.
positions Map Map of variant position -> matrix position. Both positions should be integers.
